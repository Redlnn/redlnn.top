# redlnn.top

Red_lnn 的个人主页 👉 [传送门](https://www.redlnn.top/)

基于 `Vue 3` 以及 `Element Plus`

## 部署项目

### 初始化

#### clone 本仓库

```bash
git clone git@github.com:Redlnn/redlnn.top.git
```

#### 进入本仓库本目录后执行如下命令

```bash
npm install
```

### 编译并热重载以进行开发

```bash
npm run serve
```

### 编译并最小化文件以发布到生产环境

```bash
npm run build
```

### 检查代码错误以修复

```bash
npm run lint
```
